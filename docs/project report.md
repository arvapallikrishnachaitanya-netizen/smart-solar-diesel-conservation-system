\# IoT Based Smart Diesel Conservation Using Solar Power



\## Abstract



This project is designed to reduce diesel consumption using solar energy and automation.



The system automatically switches between solar power and diesel backup depending on sunlight availability using sensors and relay control.



This project promotes renewable energy and reduces pollution caused by diesel generators.



\---



\# Introduction



Diesel generators are widely used for backup power but consume fossil fuels and produce pollution.



Solar energy is a clean and renewable energy source. By intelligently switching between solar and diesel power, fuel consumption can be reduced significantly.

\# Components Used



| Component | Purpose |

|---|---|

| Arduino UNO | Main controller |

| LDR Sensor | Detect sunlight |

| Relay Module | Switch power source |

| LCD Display | Display system status |

| LEDs | Indicate operating mode |

| Solar Panel | Renewable energy source |



\---



\# Working Principle



1\. LDR sensor detects sunlight intensity.

2\. Arduino processes sensor values.

3\. If sunlight is available:

&#x20;  - Solar mode activates

&#x20;  - Diesel backup turns OFF

4\. If sunlight is unavailable:

&#x20;  - Backup mode activates

5\. System status is displayed.
