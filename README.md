\# IoT Based Smart Diesel Conservation Using Solar Power



\## Overview



This project is designed to reduce diesel consumption using solar power and automation.



The system automatically switches between solar energy and diesel backup depending on sunlight availability.



\## Features



\- Automatic Solar/Diesel Switching

\- Diesel Conservation

\- Solar Energy Monitoring

\- Fuel Level Monitoring

\- LCD Display

\- IoT Dashboard Support



\## Components Used



\- ESP32 / Arduino UNO

\- Solar Panel

\- Relay Module

\- LDR Sensor

\- LCD Display

\- Ultrasonic Sensor

\- Battery

\- LEDs
